# spatial-trees

![spatial search](https://cdn-images-1.medium.com/max/2000/1*RsZ300nAnsPCxrd2bu5sQw.png)

Python implement for spatial trees.

The library includes following trees:

- KD-tree
- R-tree
- Ball-tree
- OCTrees
- Quadtree
- Greene-tree
- R*-tree
- X-tree
- Cover tree
